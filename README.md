# Polynomes et calculs

## Description

Les polynômes sont des objets mathématiques très utilisés dans de
 nombreux contextes. Au même titre que les nombres entiers, on peut réaliser de nombreuses
 opérations avec des polynômes : addition, soustraction, multiplication ou encore division euclidienne. On peut aussi parler du PGCD de deux polynômes.

Nous allons donc implémenter cette structure mathématique et ses opérations en langage C.
